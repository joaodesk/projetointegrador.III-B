# Projeto Integrador III-B

## Sistema Gerenciador de Tarefas - MP Pinturas

Este projeto tem como objetivo desenvolver um sistema web para gerenciamento de tarefas, aplicado à empresa MP Pinturas.

## 🔗 Protótipo no Figma
Link: https://inside-nougat-15918358.figma.site/

## 🚀 Funcionalidades
- Login de usuários
- Cadastro de projetos (obras)
- Quadro Kanban (A fazer, Em andamento, Concluído)
- Criação de tarefas
- Exclusão de tarefas e projetos (simulado)
- Visualização em calendário

## 🛠️ Tecnologias utilizadas
- Figma (prototipação)
- Trello (gestão do projeto)
- GitHub (armazenamento)

## 📌 Observação
O sistema foi desenvolvido como protótipo navegável, simulando interações reais de uso.
